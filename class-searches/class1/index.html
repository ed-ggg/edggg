Class I

Class I Green Gas Giants can vary significantly across their temperature range. They all share their usual lighter beige-white base color but can either have a dark red, pink or bright pink discoloration alongside green. A lot of Class I GGGs seem to have a significant discoloration around one or both poles, but not all have been shown to do so. Please note that a few Class I GGGs barely exhibit any green color, if any.
